# Calculator and Word counter
## main.c
A simple calculator script that takes 3 string arguments: the first operand, the operator, the second operand.  
Possible operators: ```+ * - /```
Your shell might misinterpret some of the symbols, so wrap them in quotes if not sure.

## count.c
Another simple scipt, that takes zero or one arguments, and counts the lines and words in the file.  
If zero agruments are provided it uses ```repeated.txt``` as the default.
